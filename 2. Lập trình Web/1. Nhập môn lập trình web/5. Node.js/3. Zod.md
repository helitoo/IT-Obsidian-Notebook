

[Zod](https://zod.dev/api) là một thư viện giúp kiểm tra nhanh dữ liệu (Data validation).

# Các bước sử dụng

1. **Định nghĩa schema**: Là định nghĩa của dữ liệu cần kiểm tra thông qua `z.object(...)`.
2. **Định nghĩa TypeScript**: Biến schema thành một type của TypeScript thông qua `z.infer<urSchema>`.
3. **Parse**: Phân tích và trả về lỗi.
