
Next.js có hỗ trợ sẵn việc render hình ảnh thông qua component `Image` của `next/image`.
- **Đối với các hình ảnh tại local**: Giảm kích thước ảnh về giá trị của tham số `width` và `height` của component.
- **Đối với các hình ảnh bên ngoài**: Next.js sẽ chặn chúng. Bạn có thể sửa hành vi này của Next.js thông qua `nextConfig.images.remotePatterns` trong `next.config.mjs`:

`next.config.mjs`
```mjs
const nextConfig = {
	images: {
		remotePatterns: [
			{
				protocol: 'https',
				hostname: 'assets.example.com',
				port: '',
				pathname: '/**',
			},
			// ...
		],
	}
}
```

**Component `Image` có một số tham số quan trọng sau**:
1. Tham số bắt buộc: `src`, `alt`.
2. Tham số tùy chọn: `width`, `height`. Giá trị mặc định là `500x500`.

>[!tip]
>Sau khi dùng thuộc tính `width` và `height` của `Image`, bạn **vẫn có thể chèn thêm các Tailwind class quy định kích thước như thường**.

Đọc thêm về component `Image` tại [đây](https://nextjs.org/docs/app/api-reference/components/image).
