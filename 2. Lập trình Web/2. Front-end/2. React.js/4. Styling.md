Có một số phương pháp sau khi style cho component:

# Prop

**Style** cũng tương tự như prop, cũng là một object.

VD:
```jsx
const boxStyle = {
  border: "1px solid black",
  padding: "10px",
  borderRadius: "8px",
};

<div style={boxStyle}></div>
```

# CSS module

**CSS module** là định nghĩa các style ở một file riêng, rồi nhúng vào các component cần thiết.

Các CSS sẽ được định dạng trong một file có đuôi `.module.css`. Trong file này, bạn chỉ cần khai báo các class và style theo ý muốn. Khi được gán vào JSX và biên dịch thành HTML, các class này sẽ được React biến đổi sao cho độc nhất, không trùng nhau.

VD:
`MyBox.module.css`:
```css
.box {
  border: 2px solid green;
  padding: 16px;
  border-radius: 8px;
  background-color: #f0fdf4;
}

.title {
  color: darkgreen;
  font-size: 20px;
}
```
Sử dụng CSS module:
```jsx
import styles from './MyBox.module.css';

function MyBox() {
  return (
    <div className={styles.box}>
      <h2 className={styles.title}>Tiêu đề</h2>
      <p>Đây là nội dung bên trong hộp.</p>
    </div>
  );
}
```
Khi biên dịch JSX trên lên browser, các class sẽ được tự động phân tích thành các class riêng biệt, không thể trùng nhau, như `.box` sẽ trở thành `.box__MyBox__92a7f`,... .

# Class và `clsx`

---
Cách thông dụng nhất hiện nay để styling cho component là dùng Tailwind class.

```jsx
<div className="px-4 py-2 rounded font-semibold ..."></div>
```

---
Ngoài sự đơn giản, thuận tiện của nó, Tailwind class khá tiện để tùy chỉnh class.

VD, tôi muốn component có các style khác nhau khi gặp các state khác nhau thì có thể dùng **template string và toán tử ba ngôi**.

```jsx
function Button({ isActive, isDisabled, outlined }) {
  return (
    <button
      className={`
          px-4 py-2 rounded font-semibold
          ${
              isActive ? "bg-blue-500 text-white" : "bg-gray-300 text-black"
          }
          ${isDisabled ? "opacity-50 cursor-not-allowed" : ""}
          ${outlined ? "border border-blue-500" : ""}`
      }
      disabled={isDisabled}
    >
      Click me
    </button>
  );
}
```

---
Trong trường hợp có quá nhiều điều kiện, cú pháp trên tỏ ra kém hiệu quả. Ta có thể dùng thư viện `clsx`.

Tải thư viện:
```sh
npm install clsx
```

Cú pháp cơ bản:
```jsx
import clsx from "clsx";

//
clsx(
	"Các class không thay đổi",
	{
		"Các class sinh ra khi x == true": x,
		"Các class sinh ra khi y == true": y,
		...
	}
)
```

VD:

```jsx
import clsx from "clsx";

function Button({ isActive, isDisabled, outlined }) {
  return (
    <button
      className={clsx(
        "px-4 py-2 rounded font-semibold",
        {
          "bg-blue-500 text-white": isActive,
          "bg-gray-300 text-black": !isActive,
          "opacity-50 cursor-not-allowed": isDisabled,
          "border border-blue-500": outlined,
        }
      )}
      disabled={isDisabled}
    >
      Click me
    </button>
  );
}
```

# Global CSS

Là các style chung nhất, áp dụng cho toàn bộ trang web, được định nghĩa tại `/src/app/globals.css` và import tại `App.jsx`.

Có 3 mức độ ưu tiên (layer) tại global:
1. `base`: Mức ưu tiên thấp nhất.
2. `components`: Mức ưu tiên cao hơn, ghi đè được `base`.
3. `utilities`: Mức ưu tiên cao nhất, ghi đè được các mức ưu tiên trên.

Bạn cũng có thể tự khai báo các style mà không phụ thuộc vào các layer trên.

Khai báo sử dụng các layer:
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

Khai báo nội dung layer `base`:
```css
@layer base{
	/* Your styles */
}

@layer components{
	/* Your styles */
}

@layer utilities{
	/* Your styles */
}
```
