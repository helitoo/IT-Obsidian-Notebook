
# Aurora and RDS (Relational database service)

Là dịch vụ triển khai relational database của AWS, hỗ trợ đang dạng các loại database như:
- Aurora (MySQL compatible).
- Aurora (PostgreSQL compatible).
- MySQL.
- PostgreSQL.
- MariaDB.
- Oracle.
- SQL Server.
- IBM Db2.

**Phương pháp khởi tạo database**:
- **Full config**: Config toàn quyền database.
- **Easy create**: Chỉ cấu hình thủ công các config database, còn lại do AWS lo.

**Templates**:
- **Production**: Hiệu năng cao và ổn định.
- **Dev/Test**: Đủ cho quy trình phát triển.
- **Sandbox**: 

**Availability and durability**:
- **Single-AZ instance development**: Gồm 1 instance (AZ-1) chạy ở 1 zone.
- **Multi-AZ instance development**: Gồm 2 instance (AZ-1, AZ-2), AZ-1 dành cho đọc/ghi và AZ-2 dành cho chạy dự phòng khi AZ-1 gặp sự cố.
- **Multi-AZ DB Cluster**: Gồm 3 instance (AZ-1, AZ-2, AZ-3), AZ-1 dành cho đọc ghi, AZ-2 và AZ-3 dành cho đồng bộ dữ liệu và host dự phòng.
