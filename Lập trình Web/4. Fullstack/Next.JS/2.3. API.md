# App routers

Ban đầu, Next.js sử dụng **Pages router** (`page/api/`) dựa trên [[1. Node.js|Node.js]] và [[Express JS]]. Sau này, Vercel đã giới thiệu **App router** là tinh gọn và mạnh mẽ hơn của Pages router. App router là cài đặt mặc định của Next.js.

>[!note]
>Nếu bạn chỉ cần fetch dữ liệu từ server thì Server components / Server actions là đủ, không cần xây dựng API.

# Xây dựng API

Route của API cũng tuân theo [[2.1. Routing#Routing path|routing path]] của Nextjs. Các handler của mỗi API được đặt tại `route.ts`.

Cấu trúc một handler:
```ts
export async function METHOD( request: Request ) {...}
```
Với [[9. Fetch API|Request]] là đối tượng vốn có của JS.

VD:
`app/api/users/route.ts`
```ts
export async function GET(request: Request) {
  const users = // ...
  
  return new Response(JSON.stringify(users), {
    status: 200,
    headers: { 'Content-Type': 'application/json' }
  });
}
 
export async function POST(request: Request) {
  const body = await request.json();
  const { name } = body;
 
  const newUser = { id: Date.now(), name };
 
  return new Response(JSON.stringify(newUser), {
    status: 201,
    headers: { 'Content-Type': 'application/json' }
  });
}
```

# Các cấu hình đặc biệt của routing path

## Query parameter

Lúc này, handler sẽ nhận tham số có kiểu `NextRequest` và các query parameter sẽ được truy xuất thông qua phương thức `.nextUrl.urParams.get('urParams')`.

VD:
`app/api/search/route.ts`
```ts
import { NextRequest } from 'next/server';
 
export function GET(request: NextRequest) {
  const searchParams = request.nextUrl.searchParams;
  const query = searchParams.get('query');
 
  // ...
}
```

API trên sẽ chấp nhận các path kiểu `/api/search?query=hello`.

## Dynamic routes

Lúc này, các segment sẽ trở thành tham số. Handler nhận thêm tham số `{ params }` đại diện cho các segment và được truy xuất bằng `(await params).urParams`.

VD:
`app/api/users/[id]/route.ts`
```ts
import { NextRequest } from 'next/server';
 
export async function GET(
  request: NextRequest,
  { params }: { params: Promise<{ id: string }> },
) {
  const id = (await params).id;
  
  return new Response(JSON.stringify({ id, name: `User ${id}` }), {
    status: 200,
    headers: { 'Content-Type': 'application/json' },
  });
}
```
