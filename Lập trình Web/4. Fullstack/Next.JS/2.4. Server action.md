
**Server action** là các hàm không đồng bộ (Asynchronous) tại server, có thể được gọi bởi server component và client component bất cứ lúc nào.

Server action được định nghĩa có directive `'use server'` ngay trước nội dung của hàm đó hoặc ở đầu file (Component server).

Server action chỉ có thể được gọi trực tiếp bởi **Server components và các Actions khác**.
Đối với **Client component**, phải dùng phương thức `.bind(null, ...args)` để truyền tham số.

VD:

Server component: Thu thập dữ liệu form và đưa vào database `db`:
`app/actions.ts`
```tsx
"use server";

export async function updateUserName(userId: string, formData: FormData) {
  const newName = formData.get("newName") as string;
  await db.user.update(userId, { name: newName });
}

```

Client component:
`app/profile/page.tsx`
```tsx
import { updateUserName } from "../actions";

export default function ProfilePage() {
  const userId = "abc123";

  return (
    <form action={updateUserName.bind(null, userId)}>
      <input type="text" name="newName" />
      <button type="submit">Save</button>
    </form>
  );
}

```
