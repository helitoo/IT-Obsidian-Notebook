
Next.js có hỗ trợ sẵn các font tại local thông qua `next/font`.

# `next/font/google`

Là kho chứa các font trong hệ sinh thái **Google Font**, Next.js sẽ tự tải về source khi build.

**Các bước sử dụng**:
1. Import font chữ bạn muốn (*Dưới dạng constructor*) từ `next/font/google`.
2. Tạo instance của constructor trên. Có một số tham số như:
	1. `ital`, `wght`,... tra cứu tại [đây](https://fonts.google.com/variablefonts#axis-definitions).
	2. `subsets`: Danh sách các ngôn ngữ của font.
3. Nhúng font vào TSX dưới dạng class bằng thuộc tính `.className` của instance trên.

VD:
```tsx
import { Roboto } from 'next/font/google'
 
const roboto = Roboto({
  weight: '400',
  subsets: ['latin'],
})
 
export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en" className={roboto.className}>
      <body>{children}</body>
    </html>
  )
}
```
VD này đang dùng font `Roboto`, độ dài `'400'`, ngôn ngữ `'latin'`.

# `next/font/local`

Là các font do bạn tự tải về.

**Các bước sử dụng**:
1. Đưa các file font vào `public/` hoặc `app/`.
2. Import font chữ bạn muốn (*Dưới dạng constructor*) từ `next/font/local`.
3. Tạo instance của constructor trên. Hàm này có 1 tham số quan trọng là `src`: Đường dẫn tới file font.
4. Nhúng font vào TSX dưới dạng class bằng thuộc tính `.className` của instance trên.

VD:
```tsx
import localFont from 'next/font/local'
 
const myFont = localFont({
  src: './my-font.woff2',
})
 
export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en" className={myFont.className}>
      <body>{children}</body>
    </html>
  )
}
```

Bạn cũng có thể khai báo một font family:
```tsx
const roboto = localFont({
  src: [
    {
      path: './Roboto-Regular.woff2',
      weight: '400',
      style: 'normal',
    },
    {
      path: './Roboto-Italic.woff2',
      weight: '400',
      style: 'italic',
    },
    // ...
  ],
})
```

# Font variable

Bạn có thể đặt tên của font thông qua thuộc tính `variable` của construct, sau đó dùng lại ở bất kỳ đâu.

VD:
```tsx
const roboto = Roboto({
  // ...
  variable: '--font-roboto', // Naming the font
})
 
export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en" className={roboto.variable}>
      <body>{children}</body>
    </html>
  )
}
```

Trong `tailwind.config.ts`, tại `config.theme.extend.fontFamily`: Điền tên variable vào:

Đọc thêm tại [đây](https://tailwindcss.com/docs/font-family).
