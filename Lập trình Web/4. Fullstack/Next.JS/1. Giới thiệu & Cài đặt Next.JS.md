# Giới thiệu

**Next.JS** là một web framework dựa trên [[1. Giới thiệu & Cài đặt ReactJS|ReactJS]] và bổ sung các thế mạnh của SSR.

# Cài đặt

Dự án Next.js có thể được khởi tạo nhanh thông qua lệnh:
```sh
npx create-next-app@latest
```

Bạn có thể tải kèm Shadcn UI bằng:
```sh
npx shadcn@latest init
```

Một số cài đặt khi khởi tạo dự án:
```sh
What is your project named? your-app-name
Would you like to use the recommended Next.js defaults?
    Yes, use recommended defaults - TypeScript, ESLint, Tailwind CSS, App Router, Turbopack
    No, reuse previous settings
    No, customize settings - Choose your own preferences
```

# Cấu trúc dự án Next.js

Gồm có một số thư mục quan trọng sau:
1. `.next/`: Chứa các cấu hình Next.JS server, được tạo ra khi build project.
2. `node_modules/`: Chứa các thư viện của Node.JS, được tạo ra khi build project.
3. `public/`: Chứa các tài nguyên công khai.
4. `src/`: Chứa mã nguồn.
5. `env/`: Chứa cấu hình API.

Nguyên lý hoạt động của Next.js cũng tương tự ReactJS, chuỗi bài viết này chỉ nói về các điểm khác.
