# TailwindCSS Library

![](https://upload.wikimedia.org/wikipedia/commons/thumb/0/06/Tailwind_CSS_logo_with_dark_text.svg/2560px-Tailwind_CSS_logo_with_dark_text.svg.png)

Nhìn chung, ta có 2 cách để nhúng CSS vào HTML:
1. Nhúng trực tiếp vào HTML element, tuy trực quan nhưng code dài, khó quản lý.
2. Viết toàn bộ CSS ra ngoài rồi nhúng vào HTML sau, nhưng phải định nghĩa các selector phức tạp.

**Tailwind** là một thư viện, có thể nói là đã khắc phục được các điểm yếu trên của CSS truyền thống:
1. Tailwind cung cấp hệ thống các class, mỗi class tương ứng với một CSS style nhất định.
2. Muốn HTML element nào có style nào thì chỉ cần viết class, không cần nhúng CSS vất vả như thường.
3. Hầu hết các cấu hình CSS đã được chuyển hóa thành các Tailwind class.

Trong chuỗi bài viết nói về [[1. CSS]], chúng ta đã tìm hiểu về các class này.
