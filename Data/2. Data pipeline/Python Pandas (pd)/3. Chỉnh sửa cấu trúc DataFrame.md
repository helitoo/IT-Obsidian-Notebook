DataFrame có thể được lọc có điều kiện bằng [[2. Khảo sát Series và DataFrame|các phương pháp truy xuất Serie và DataFrame]]. Ngoài ra còn các phương pháp thêm / xóa vô điều kiện như sau:

# Đổi tên index và column

```python
df.rename()
```
Rename có 2 loại tham số:
- `columns={oldName: newName, ...}`: Đổi tên các cột.
- `index={oldName: newName, ...}`: Đổi tên các hàng.

```python
df.rename_axis(name, axis='index')
```
Đổi tên trục.
- `name`: Tên mới.
- `axis='index'|'columns'`.

Ngoài ra có các hàm như `df.rename_axis()` hoặc `df.index.name` cũng tương tự.

# Thêm

```python
df['newFieldName'] = ...
```
Thêm một field mới có tên là `newFieldName`.

```python
pandas.concat([df1, df2], ignore_index=True);
```
Thêm DataFrame `df2` vào cuối `df1` theo cột.

```python
df1.join(df2, on='index', how='left');
```
Thêm DataFrame `newDf` vào cuối `df` theo hàng.
Tham số:
- `on`:
	- Cột chung để ghép 2 DataFrame.
	- Nếu không có cột chung (tên cột khóa chính khác tên cột khóa ngoại) thì dùng 2 thuộc tính `left_join` và `right_join` thay thế.
	- Nếu chỉ đơn thuần ghép theo hàng, không cần quan hệ khóa ngoại, khóa nội thì dùng `left_index=True` và `right_index=True` thay thế.
- `how`: `left` join, `right` join, `inner` join và `outer` join.
- Nếu DataFrame được tạo ra chứa các cột trùng tên ở 2 DataFrame nguồn thì ta có thể đổi tên nhanh bằng thuộc tính `rsuffix` (hậu tố cột trùng tên của DataFrame bên trái), `rinffix` (tiền tố cột trùng tên của DataFrame bên trái), tương tự với `lsuffix` và `linfix`.

```python
pd.merge(df1, df2, on='index')
```
Là sự viết tắt của `df1.join(df2, on='?', how='inner')`.

# Xóa

```python
df.drop();
```
Hàm này khá đa năng, nó có một số tham số sau:
- `inplace=False`: Thể hiện rằng hàm sẽ sao chép lại `df`, thực hiện làm sạch rồi trả về kết quả (tham trị).
- `columns` (`str`|`[str]`): Tên của field cần xóa hoặc mảng các field cần xóa.
- `index` (`int`|`[int]`): Index cần xóa hoặc mảng các index cần xóa.
