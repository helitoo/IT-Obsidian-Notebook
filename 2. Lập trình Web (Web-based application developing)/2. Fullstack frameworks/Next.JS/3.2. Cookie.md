Trong Next.js, cookie là một hàm bất đồng bộ (*Async function*) dùng để đọc / ghi cookie.

>[!note]
>Có nhiều cookie, mỗi cookie có dạng `'name': value`.

Có một số phương thức sau:

| Phương thức                  | Kiểu             | Mô tả                                    |
| ---------------------------- | ---------------- | ---------------------------------------- |
| `.get('name')`               | Object           | Trả về giá trị cookie `name`.            |
| `.getAll()`                  | Array of objects | Trả về danh sách các cookie.             |
| `.has('name')`               | Boolean          | Trả về `true` nếu tồn tại cookie `name`. |
| `.set(name, value, options)` |                  | Lưu cookie `name` với giá trị `value`.   |
| `.delete(name)`              |                  | Xóa cookie `name`.                       |
| `.clear()`                   |                  | Xóa mọi cookie.                          |
| `.toString()`                | String           | Chuyển cookie thành string.              |

Khi set cookie, có một số option sau:

| Option        | Kiểu                                   | Mô tả                                                                                                                                                                                         |
| ------------- | -------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `expires`     | Date                                   | Thời điểm cookie hết hạn.                                                                                                                                                                     |
| `maxAge`      | Number                                 | Thời gian cookie còn hạn (giây).                                                                                                                                                              |
| `domain`      | String                                 | Tên miền mà cookie có hiệu lực.                                                                                                                                                               |
| `path`        | String, default: `'/'`                 | Routing path mà cookie có hiệu lực.                                                                                                                                                           |
| `secure`      | Boolean                                | Chỉ cho phép dùng cookie với protocol `https`.                                                                                                                                                |
| `httpOnly`    | Boolean                                | Chỉ cho phép server dùng cookie.                                                                                                                                                              |
| `sameSite`    | Boolean, `'lax'`, `'strict'`, `'none'` | - `lax`: Cookie chỉ có hiệu lưc khi cùng domain.<br><br>- `strict`: Cookie có hiệu lực trên nhiều domain nhưng chỉ khi thông qua các phương thức an toàn như link hay `GET`.<br><br>- `None`. |
| `priority`    | `"low"`, `"medium"`, `"high"`          | Mức độ ưu tiên giữ cookie lại (*trong trường hợp browser xóa các cookie tốn bộ nhớ*).                                                                                                         |

**VD: Tạo cookie khi user login**:
```tsx

import { cookies } from 'next/headers'

export async function loginAction(formData: FormData) {
  const username = formData.get('username')
  const password = formData.get('password')
  
  if (is_valid(username, password)) {
    const token = 'abc123xyz'
    
    // Tạo cookie đăng nhập
    const cookieStore = await cookies()
    
    cookieStore.set('auth_token', token, {
      httpOnly: true,
      secure: process.env.NODE_ENV === 'production',
      sameSite: 'lax',
      path: '/',
      maxAge: 60 * 60 * 24 * 7, // 7 ngày
    })

    return { success: true }
  }

  return { success: false, error: 'Sai thông tin đăng nhập' }
}
```

**VD: Dùng cookie để hiển thị thông tin client**:
```tsx
import { cookies } from 'next/headers'
import { redirect } from 'next/navigation'

export default async function DashboardPage() {
  const cookieStore = await cookies()
  const token = cookieStore.get('auth_token')?.value

  if (!token)
    redirect('/login')

  return (
    <main className="p-5">
      <h1 className="text-2xl font-bold">Xin chào, Admin 👋</h1>
      <p>Token hiện tại: {token}</p>
    </main>
  )
}
```

**VD: Xóa cookie khi đăng xuất**:
```tsx
'use server'

import { cookies } from 'next/headers'

export async function logoutAction() {
  const cookieStore = await cookies()
  cookieStore.delete('auth_token')
}
```
