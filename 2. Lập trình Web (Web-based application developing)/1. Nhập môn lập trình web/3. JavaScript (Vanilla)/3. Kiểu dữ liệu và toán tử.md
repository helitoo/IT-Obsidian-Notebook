# Data Types

## Primitive types

Primitive là các kiểu dữ liệu cơ bản, lưu trữ trực tiếp giá trị.

|Type|Ví dụ|
|---|---|
|Number|`10`, `3.14`|
|String|`"Hello"`|
|Boolean|`true`, `false`|
|Undefined|`undefined`|
|Null|`null`|
|BigInt|`123n`|
|Symbol|`Symbol("id")`|

Lưu ý rằng `typeof null` là `object` chứ không phải `null`.

Các primitive đều là **immutable** (không thể thay đổi giá trị), cho nên nó **pass-by-value**: Khi gán 1 giá trị cho 1 biến thì nó copy giá trị đó sang vùng nhớ mới rồi gán vào biến chứ không tham chiếu.
```js
let s = "abcde";
s[0] = "f"; // Vô hiệu
```

**Type coercion**: Cơ chế tự động ép kiểu trong các biểu thức thành kiểu khả dĩ khi thực hiện biểu thức trong JS:
```js
"5" + 3 // "53"
"5" - 3 // 2
[] == false // tương đương "" == 0, tương đương 0 == 0, và là true
```
-> Khuyến khích dùng:
- Strict equality `===` để kiểm tra cả giá trị và kiểu dữ liệu để tránh các lỗi không đáng có.
- Ép kiểu tường minh (explicit) như `Number()`, `String()`, `Boolean()`,...

**Falsy**: Là các giá trị tương đương với `false` trong biểu thức logic: `false`, `0`, `-0`, `0n`, `""`, `null`, `undefined`, `NaN`.
**Truthy**: Là mọi giá trị khác falsy, kể cả `"0"`, `[]`, `{}`.
-> Khuyến khích dùng `Boolean()` hoặc `!!v` để kiểm tra giá trị Boolean tương ứng của nó.
-> Lưu ý là falsy hay truthy không phải là kết quả sau khi ép kiểu (type coercion).

## Object types

Object là các kiểu dữ liệu có thể chứa nhiều thuộc tính hoặc phương thức.

Các primitive đều là **mutable** (có thể thay đổi giá trị), cho nên nó **pass-by-reference** (tham chiếu), trái ngược với primitive. Khi 1 biến `a` được gán cho 1 biến `b`, đều là object, thì bản chất 2 biến này đều trỏ tới 1 địa chỉ, khi thay đổi giá trị object qua 1 biến bất kỳ thì biến còn lại cũng nhận được sự thay đổi đó.

**Object**:
```js
const student = {
    name: "An",
    age: 20
};

console.log(student.name);
```

**Array**:
```js
const scores = [8, 9, 10];

console.log(scores[0]);
```

**Map**:
```js
const scores = new Map();

scores.set("An", 9);
scores.set("Binh", 8);

console.log(scores.get("An"));
```

**Function**: Xem kỹ hơn trong phần [[2. Cú pháp cơ bản của JS]].
```js
function greet(name) {
    return `Hello ${name}`;
}

console.log(greet("An"));
```

## Classes

Class là một **khuôn mẫu (blueprint)** dùng để tạo ra nhiều object có cấu trúc giống nhau.

JS thực chất là ngôn ngữ **prototype-based**, không phải class-based như Java. Nên các object có thể được tạo ra tự do mà không cần xây dựng class từ trước.

>[!quote]
>**Class không thêm khả năng mới cho JS**, mà chủ yếu giúp code **dễ đọc, dễ viết, dễ bảo trì** hơn.

**VD**: Xây dựng class, thân thiện với cú pháp OOP truyền thống.
```js
class Student {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    
    sayHello() {
        console.log("Hello");
    }
}

// Khai báo instance
const s1 = new Student("An", 18);
const s2 = new Student("Bình", 19);

console.log(s1.name); // An
console.log(s2.name); // Bình
```

**VD**: Xây dựng prototype thủ công, thuần túy bằng JS nhưng khá khó đọc.
```js
function Student(name, age) {
    this.name = name;
    this.age = age;
}

Student.prototype.sayHello = function() {
    console.log("Hello");
};

const s1 = new Student("An", 18);
const s2 = new Student("Bình", 19);
```

# Operators

## Arithmetic operators

Dùng để thực hiện các phép toán số học.

|Toán tử|Ý nghĩa|
|---|---|
|`+`|Cộng|
|`-`|Trừ|
|`*`|Nhân|
|`/`|Chia|
|`%`|Chia lấy dư|
|`**`|Lũy thừa|

## Assignment operators

Dùng để gán giá trị.

```js
let x = 10;

x += 5; // x = x + 5
x -= 2;
x *= 3;
x /= 2;
x ++; ++ x; // Tương tự C++
x --; -- x;
```

## Comparison operators

Dùng để so sánh giá trị.

| Toán tử | Ý nghĩa                         |
| ------- | ------------------------------- |
| `==`    | So sánh giá trị                 |
| `===`   | So sánh giá trị và kiểu dữ liệu |
| `!=`    | Khác giá trị                    |
| `!==`   | Khác giá trị hoặc kiểu dữ liệu  |
| `>`     | Lớn hơn                         |
| `<`     | Nhỏ hơn                         |
| `>=`    | Lớn hơn hoặc bằng               |
| `<=`    | Nhỏ hơn hoặc bằng               |

```js
console.log(5 == "5");   // true
console.log(5 === "5");  // false
```

## Logical operators

Dùng để kết hợp nhiều điều kiện.

| Toán tử | Ý nghĩa |
| ------- | ------- |
| `&&`    | AND     |
| \|\|    | OR      |
| `!`     | NOT     |

```js
!value   // đảo ngược giá trị truthy/falsy
!!value  // đảo ngược lần nữa -> boolean tương ứng

console.log(!!1);        // true
console.log(!!0);        // false

console.log(!!"Hello");  // true
console.log(!!"");       // false

console.log(!![]);       // true
console.log(!!{});       // true

console.log(!!null);     // false
console.log(!!undefined);// false
```

## Spencial operators

**Ternary operator**: Viết ngắn gọn cho cấu trúc *if-else*.

```js
// condition ? value1 : value2;

let age = 20;
let result = age >= 18 ? "Adult" : "Child";
```

**Short-Circuit evaluation**:
- `a && b`: Trả về toán hạng đầu tiên là falsy, hoặc toán hạng cuối cùng nếu tất cả đều truthy.
- `a || b`: Chỉ trả về 1 trong 2 nếu cái còn lại là falsy.
- `a ?? b`: Chỉ trả về 1 trong 2 nếu cái còn lại là `null`/`undefined`.

```js
let score = 0;  
  
console.log(score || 100); // 100  
console.log(score ?? 100); // 0
```
