
# Branching

Cũng tương tự hầu hết các ngôn ngữ lập trình khác.

**VD1**: If-Else:
```js
let score = 85;

if (score >= 90) { console.log("A"); }
else if (score >= 80) { console.log("B"); }
else if (score >= 70) { console.log("C"); }
else { console.log("D"); }
```

**VD2**: Switch-case:
```js
let day = 2;

switch (day) {
    case 1:
        console.log("Monday");
        break;
        
    case 2:
        console.log("Tuesday");
        break;
        
    default:
        console.log("Unknown");
}
```


# Loop

## For-based

**For**:
```js
for (let i = 1; i <= 5; i++) {
	console.log(i);
}
```

**For-of**: Duyệt qua từng value của array, không thông qua index.
```js
// Array
const arr = [10, 20, 30];

for (const value of arr) {
    console.log(value);
}

// Map
const scores = new Map([
    ["Alice", 90],
    ["Bob", 85],
    ["Charlie", 95]
]);

for (const [key, value] of scores) {
    console.log(key, value);
}
```

**ForEach**: Cũng giống *for-of*, nhưng sử dụng arrow-function:
```js
arr.forEach(value => {
    console.log(value); // return sẽ tương tự continue
});
```

**For-in**: Duyệt qua từng key của object.
```js
const student = {
    name: "An",
    age: 20
};

for (const key in student) {
    console.log(key, student[key]);
}
```

## While-based

**While-do**:
```js
let i = 1;

while (i <= 5) {
    console.log(i);
    i++;
}
```

**Do-while**: Thực hiện ít nhất 1 lần.
```js
let i = 10;

do {
	console.log(i);
	i++;
} while (i < 5);
```

# Timer

Timer là các hàm dùng để **hẹn giờ hoặc lặp lại hành động sau một khoảng thời gian**.

## Set time out - Thực hiện một lần

```js
setTimeout(urFunc, delay);
```
`urFunc`: Hàm được kích hoạt **sau khi `delay` milisecond**.

## Set Interval - Thực hiện theo chu kỳ

```js
setInterval(urFunc, delay);
```
`urFunc`: Hàm được kích hoạt **sau mỗi `delay` milisecond**. Nếu thời gian thực thi `urFunc` lâu hơn `delay` thì các lần gọi hàm sau sẽ được đưa vào queue.

```js
const timer = setInterval(urFunc, delay);
//
clearInterval(timer);
```
`clearInterval(...)` để hủy vòng lặp của `setInterval`.

>[!note]
>Cấu trúc lặp kết hợp với `setTimeOut` sẽ trở thành `setInterval`.
