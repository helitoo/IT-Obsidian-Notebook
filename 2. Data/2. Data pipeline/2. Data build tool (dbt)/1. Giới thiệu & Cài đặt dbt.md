# Giới thiệu

**dbt** là một công cụ mã nguồn mở hỗ trợ **xử lý dữ liệu**, tiếp cận từ hướng **data analisis**, tức là sử dụng **SQL** làm khuông mẫu.
- **dbt Core**: Một dự án mã nguồn mở hỗ trợ các thao tác với một dbt project (*`.sql`, `.yml`,...*).
- **dbt Cloud**: Nền tảng SaaS.
