# Missing values

## Kiểm tra missing value

Các data point bị lỗi sẽ được Pandas gắn giá trị là `NaN`.

Kiểm tra số lượng missing value:
```python
# get the number of missing data points per column
missing_values_count = nfl_data.isnull().sum()

# look at the # of missing points in the first ten columns
missing_values_count[0:10]
```

Kiểm tra tỷ lệ số lượng missing value:
```python
# how many total missing values do we have?
total_cells = np.product(nfl_data.shape)
total_missing = missing_values_count.sum()

# percent of data that is missing
percent_missing = (total_missing/total_cells) * 100
```

## Nguyên nhân và khắc phục missing

| Tình huống                                 | Hướng giải quyết                                                         |
| ------------------------------------------ | ------------------------------------------------------------------------ |
| Dữ liệu không được ghi nhận trong dataset. | - Xóa các record/column missing.<br>- Điền giá trị vào các missing cell. |
| Dữ liệu vốn không có.                      | Giữ nguyên `NaN` hoặc biến đổi thành các dạng khác cho tường minh hơn.   |
Kiểm tra các cột bị miss:

```python
# Get names of columns with missing values
missing_cols = [col for col in X_train.columns if X_train[col].isnull().any()]
```

#### Remove missing columns/records

- Dùng khi các missing khá ít, không quan trọng, hoặc không cần thiết để tìm hiểu lý do missing.
- Phương pháp này không được khuyến khích vì nó làm mất đi tính toàn vẹn của dữ liệu.

```python
# remove all the rows that contain a missing value
X_train.dropna()
```

Việc xóa các record dễ dẫn đến bay màu toàn bộ dataset.
Thay vào đó, nên xóa các column:
```python
# remove all columns with at least one missing value
dropped_cols = X_train.dropna(axis=1)
# X_train.drop(missing_cols, axis=1)
```

#### Simple imputing

**Điền cứng giá trị vào missing cell**:
```python
# replace all NA's with 0
subset_nfl_data.fillna(0)
```

**Lấy giá trị ở record phía dưới (`bfill`) (nếu có), nếu không thì điền cứng**:
```python
subset_nfl_data.fillna(method='bfill', axis=0).fillna(0)
```

**Lấy các chỉ báo thống kê để điền**:
```python
from sklearn.impute import SimpleImputer

# Using 'mean' for imputating
my_imputer = SimpleImputer()
# Calculating 'mean' and transforming
imputed_X_train = pd.DataFrame(my_imputer.fit_transform(X_train))

imputed_X_train.columns = X_train.columns
```

VD:

```python
landslides = pd.read_csv("../input/landslide-events/catalog.csv")
landslides['date'].dtype
```
```sh
dtype('O')
```

Như bạn thấy, Pandas nhận diện `date` có kiểu `object`, thay vì `datetime64`.

Chuyển `date` sang kiểu `datetime64`:
```python
landslides['date_parsed'] = pd.to_datetime(landslides['date'], format="%m/%d/%y")
```

Tra cứu danh sách format (*strftime*) [tại đây](https://strftime.org/). Có một số ký hiệu cần nhớ:
- `%m`: Month.
- `%d`: Day.
- `%y`: Year (2 chữ số cuối).
- `%Y`: Year (4 chữ số).

Pandas hỗ trợ các phương thức sau đối với `datetime64`:
- `.dt.day`: Lấy ngày.
- Tương tự với `.dt.month`, `.dt.year`, `.dt.weekend`,...

Có thể thấy, có rất nhiều biến thể của `Germany` như `germany`, hay `New Zealand` và `New Zealand`.

Có một số hướng giải quyết sau:

**Thống nhất định dạng (chữ in hoa, in thường, dấu câu,...)**:
```python
# convert to lower case
professors['Country'] = professors['Country'].str.lower()
# remove trailing white spaces
professors['Country'] = professors['Country'].str.strip()
```

**Chuẩn hóa bằng fuzy search**: Fuzy search là tìm kiếm xấp xỉ - tìm kiếm đáp án khớp nhất thay vì tìm kiếm đáp án khớp hoàn toàn.

Các bước chuẩn hóa chuỗi `string_to_match`:
1. Trích xuất các giá trị khác nhau hiện có (`.unique`).
2. Chọn ra chuỗi khớp nhất với `string_to_match` dựa trên các giá trị khác nhau như trên.
3. Thay thế toàn bộ `string_to_match` bằng chuỗi khớp nhất vừa tìm thấy.

Trên thực tế, có thể chèn thêm ràng buộc `min_ratio`: Chuỗi khớp nhất phải khớp trên ngưỡng `min_ratio` thì mới được chấp nhận.

```python
from fuzzywuzzy import process

# function to replace rows in the provided column of the provided dataframe
# that match the provided string above the provided ratio with the provided string
def replace_matches_in_column(df, column, string_to_match, min_ratio = 47):
    # get a list of unique strings
    strings = df[column].unique()
    
    # get the top 10 closest matches to our input string
    matches = fuzzywuzzy.process.extract(string_to_match, strings, 
                                         limit=10, scorer=fuzzywuzzy.fuzz.token_sort_ratio)
	
    # only get matches with a ratio > 90
    close_matches = [matches[0] for matches in matches if matches[1] >= min_ratio]
	
    # get the rows of all the close matches in our dataframe
    rows_with_matches = df[column].isin(close_matches)
	
    # replace all rows with close matches with the input matches 
    df.loc[rows_with_matches, column] = string_to_match
    
    # let us know the function's done
    print("All done!")


replace_matches_in_column(df=professors, column='Country', string_to_match="south korea")
```
