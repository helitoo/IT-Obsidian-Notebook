**API (Application Programming Interface)** là giao diện để kết nối các ứng dụng với nhau. **RESTful API** là một chuẩn API được thiết kế để chuẩn hóa và tối ưu trong việc xây dựng web.

**REST (Repretational State Transfer)** là một dạng chuyển đổi cấu trúc dữ liệu, một kiểu kiến trúc để viết API. Nó sử dụng **phương thức HTTP (HTTP method)** để tạo cho giao tiếp giữa các máy.

# HTTP Method

Có 4 phương thức thông dụng:
- `GET` (`SELECT`): Trả về resource.
- `POST` (`CREATE`): Tạo mới resource.
- `PUT` (`UPDATE`): Cập nhật thông tin cho resource.
- `DELETE` (`DELETE`): Xoá resource.
