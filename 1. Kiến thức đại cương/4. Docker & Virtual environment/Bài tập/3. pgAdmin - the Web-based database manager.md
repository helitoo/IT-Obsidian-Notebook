
Trong 2 bài trước ta đã tìm hiểu về `pgcli` là một CLI-based database manager. Trong bài viết này ta sẽ tìm hiểu về `pgAdmin` - quản lý database bằng giao diện web.

**1 - Tạo pg-network**:
```sh
docker network create pg-network
```
- Network có tác dụng giúp PostgreSQL container và pgadmin container giao tiếp được với nhau.
- Sau khi dùng bạn có thể xóa đi bằng `docker network rm pg-network`.

**2 - Tạo PosgreSQL container**:
```powershell
docker run -it --rm -e POSTGRES_USER=root -e POSTGRES_PASSWORD=root -e POSTGRES_DB=ny_taxi -v ny_taxi_postgres_data:/var/lib/postgresql/data -p 5432:5432 postgres:15
```

**3 - Tạo pgAdmin container**:
```powershell
docker run -it -e PGADMIN_DEFAULT_EMAIL="admin@admin.com" -e PGADMIN_DEFAULT_PASSWORD="root" -v pgadmin_data:/var/lib/pgadmin -p 8085:80 --network=pg-network --name pgadmin dpage/pgadmin4
```

Bây giờ, bạn có thể truy cập vào `http://localhost:8085`, đăng nhập và sử dụng pgadmin.
