
# Venv và Docker

**Venv** là môi trường ảo, tách biệt các module của Python bên trong venv và bên ngoài venv (host machine).
- **Venv**: Cô lập ở *mức module*, vẫn dùng chung kernel, system library,... của host machine.
- **Docker**: Cô lập ở *mức OS*, có system library riêng, user riêng,... Python chỉ là một phần của Docker.

# Cài đặt venv

\[*Các hướng dẫn sau hoạt động trên Windows OS*]

|                          | pip                               | uv                          |
| ------------------------ | --------------------------------- | --------------------------- |
| Khởi tạo venv `venv`     | `python -m venv .venv`            | `uv venv`                   |
| Kích hoạt `venv`         | `source .venv/bin/activate`       | --                          |
| Thêm package vào `venv`  | `pip install pandas`              | `uv add pandas`             |
| Thực thi `pipeline.py`   | `python pipeline.py`              | `uv run python pipeline.py` |
| Danh sách package        | `requirements.txt`                | `uv.lock`                   |
| Cài package từ danh sách | `pip install -r requirements.txt` | `uv pip sync`               |


Cài đặt uv:
```sh
pip install uv
```

Chú ý: File `uv.lock` được uv:
- Tự động tạo ra khi thêm package.
- Compile thủ công từ file `.toml`: `uv pip compile pyproject.toml`.
- uv nhận diện project là folder chứa file `pyproject.toml`.

Khởi tạo project Python nhanh bằng uv:
```sh
uv init my_project
```
