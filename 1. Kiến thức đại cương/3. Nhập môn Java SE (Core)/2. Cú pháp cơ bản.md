Các điểm sau đây khác với C++:

**Biến local** luôn phải có giá trị khởi tạo.

# Các kiểu primitive

Các kiểu giống C++: `byte`, `char`, `short`, `int`, `long`, `float`, `double`.
Các kiểu khác C++: `boolean`.

**Chú ý**: Mặc định, các giá trị số nguyên lớn hơn `byte`, `char`, `short` đều là `int`. Nếu bạn muốn chỉ thị rõ kiểu thì phải thêm các hậu tố sau (*viết hoa hay thường đều được*):
- Giá trị kiểu `long` phải kết thúc bằng `l`.
- Giá trị kiểu `float` phải kết thúc bằng `f`.
- Giá trị kiểu `double` phải kết thúc bằng `d`.
- Kiểu `boolean` không được coi như số nguyên.

# Các kiểu object

## Chuỗi ký tự `String`

Được biểu diễn trong cặp dấu **`"`**.
String có thể được cộng với String, các kiểu số học và `boolean`.

```java
String s = "This is a string";
```

**Một số phương thức**:
- `s.length()`: Độ dài `s`.
- `s.charAt(i)`: Ký tự tại index `i`.
- `s.indexOf(x)`: Index vị trí đầu tiên xuất hiện `x` trong `s`, nếu không có thì trả về `-1`.

## Mảng (Array)

**Mảng 1 chiều**:
```java
dataType[3] arr = {a, b, c};      // Khai báo với giá trị mặc định
dataType[] arr = new dataType[3]; // Khai báo mảng rỗng
```

**Mảng 2 chiều**:
```java
dataType[][] arr // = {{a, b, ...}, {c, d, ...}, ...};
```

**Một số thuộc tính & phương thức**:
- `a.length`: Độ dài `a`.
- `a[i]`: Phần tử tại index `i`.
- `a[i][j]`: Phần tử tại index `j` của mảng `i`.

**Cấu trúc duyệt mảng**: Cũng dùng `while` hoặc `for` giống C++.

**VD**: For-each:
```java
for (int cell : intArr)
	// ...
```

**VD**: For-each:
```java
for (int[] row : intArr2d)
	for (int cell : row)
		// ...
```
