**Biến local** luôn phải có giá trị khởi tạo.

# Các kiểu primitive

```
java.lang.Object
 ├── java.lang.Number
 │   ├── Byte        ← wrapper của byte
 │   ├── Short       ← wrapper của short
 │   ├── Integer     ← wrapper của int
 │   ├── Long        ← wrapper của long
 │   ├── Float       ← wrapper của float
 │   └── Double      ← wrapper của double
 ├── Character       ← wrapper của char
 └── Boolean         ← wrapper của boolean
 │
 └── java.util.Arrays
     ├── .sort(arr) (tăng dần)
     ├── .binarySearch(arr, x)
     └── .toString(arr)
```

Các kiểu giống C++: `byte`, `char`, `short`, `int`, `long`, `float`, `double`.
Các kiểu khác C++: `boolean`.

**Chú ý**: Mặc định, các giá trị số nguyên lớn hơn `byte`, `char`, `short` đều là `int`. Nếu bạn muốn chỉ thị rõ kiểu thì phải thêm các hậu tố sau (*viết hoa hay thường đều được*):
- Giá trị kiểu `long` phải kết thúc bằng `l`.
- Giá trị kiểu `float` phải kết thúc bằng `f`.
- Giá trị kiểu `double` phải kết thúc bằng `d`.
- Kiểu `boolean` không được coi như số nguyên.

# Các kiểu object

## Wrapper class

Các primitive không phải object.
Wrapper class là các lớp *hỗ trợ thao tác với biến primitive như là một object*.

Các wrapper class đều được định nghĩa trong package `java.lang` nên không cần import gì thêm.

## Chuỗi ký tự `String`

Được biểu diễn trong cặp dấu **`"`**.
String có thể được cộng với String, các kiểu số học và `boolean`.

```java
String s = "This is a string";
```

**Một số phương thức**:
- `x.length()`: Độ dài `y`.
- `x.charAt(i)`: Ký tự tại index `i` (`char`).
- `x.indexOf(y, s)`: Index vị trí đầu tiên xuất hiện `y` trong `x` bắt đầu từ `s`, nếu không có thì trả về `-1`. Mặc định, `s = 0`.
- `x.substring(a, b)`: Cắt chuỗi từ index `a` đến `b - 1`. Mặc định, `b` = `s.length()`.
- `x.equals(y)`: Trả về `true` nếu `x` = `y` (*phân biệt* chữ hoa / thường).
- `x.equalsIgnoreCase(y)`: Trả về `true` nếu `x` = `y` (*không* phân biệt chữ hoa / thường).

>[!important] Để so sánh 2 chuỗi
>- Dùng `.equals` hoặc `.equalsIngnoreCase`.
>- Toán tử `==` đối với chuỗi (và các object) là *so sánh địa chỉ được tham chiếu*, không phải so sánh giá trị.

Tìm tất cả vị trí mà `y` xuất hiện trong `x`:
```java
int index = x.indexOf(y);

if (index == -1) {
	System.out.println("x doesn't exists on y.");
} else {
	while (index != -1) {
		System.out.print(index + " ");
		index = x.indexOf(y, index + 1);
	}
}
```

## Mảng (Array)

**Mảng 1 chiều**:
```java
dataType[3] arr = {a, b, c};      // Khai báo với giá trị mặc định
dataType[] arr = new dataType[3]; // Khai báo mảng rỗng
```

**Mảng 2 chiều**:
```java
dataType[][] arr // = {{a, b, ...}, {c, d, ...}, ...};
```

Trong đó thì `dataType` vừa có thể là primitive, vừa có thể là object.
```java
int[] a;
Integer[] b; // Tốn bộ nhớ hơn nhưng dùng được một số phương thức có sẵn
```

**Một số thuộc tính & phương thức**:
- `a.length`: Độ dài của `a`.
- `a[i]`: Phần tử tại index `i`.
- `a[i][j]`: Phần tử tại index `j` của mảng `i`.
- `Arrays.toString(a)`: Chuyển thành chuỗi, thường dùng để in ra.
- `Arrays.sort(a);`: Sắp xếp theo thứ tự tăng dần.
- `Arrays.sort(a, Collections.reverseOrder());`: Sắp xếp theo thứ tự giảm dần, chỉ dùng được cho mảng của các object.
- `Arrays.binarySearch(a, x)`: Tìm vị trí `x` trong `a`, `a` đã được sắp xếp tăng dần từ trước. Nếu không tìm thấy thì trả về `-1`.
- `Arrays.copyOf(a, l)`: Trả về bản sao `l` phần tử đầu tiên của `a`. Nếu `l` > `a.length` thì bù vào những phần tử `0` (kiểu số) / `false` (kiểu boolean) / `null` (kiểu ký tự) cuối cùng.

Đảo ngược mảng:
```java
for (int i = 0; i < a.length / 2; i++) {
    int temp = a[i];
    a[i] = a[a.length - 1 - i];
    a[a.length - 1 - i] = temp;
}
```

**Cấu trúc duyệt mảng**: Cũng dùng `while` hoặc `for` giống C++.

**VD**: For-each:
```java
for (int cell : intArr)
	// ...
```

**VD**: For-each:
```java
for (int[] row : intArr2d)
	for (int cell : row)
		// ...
```
