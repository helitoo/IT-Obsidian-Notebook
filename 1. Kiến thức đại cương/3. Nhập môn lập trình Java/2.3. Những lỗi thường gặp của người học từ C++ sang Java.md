
# Biểu diễn String

```java
// ❌ C++
'A' // char
"A" // string

// ✅ Java
"A" // String
```

# So sánh String

```java
// ❌ C++ tư duy — so sánh địa chỉ, không phải nội dung
if (str == "hello")

// ✅ Java đúng
if (str.equals("hello"))
if (str.equalsIgnoreCase("hello")) // không phân biệt hoa thường
```

# Khi khai báo collection phải luôn dùng wrapper class

```java
// ❌ Java không cho dùng primitive trong generics
ArrayList<int> list;
ArrayList<double> list;

// ✅ Phải dùng Wrapper class
ArrayList<Integer> list;
ArrayList<Double> list;
```

# Khi khai báo collection phải có giá trị khởi tạo

```java
// ❌ Quên new, hoặc thiếu import
ArrayList<String> list;      // chưa khởi tạo → NullPointerException
list.add("abc");

// ✅
import java.util.ArrayList;
ArrayList<String> list = new ArrayList<String>();
ArrayList<String> list = new ArrayList<>(); // Ký hiệu diamond <>
```

# `printf` và `println`

- `println`: In ra 1 giá trị và xuống dòng.
- `printf`: Hỗ trợ format.

```java
// ❌ println không hỗ trợ format string kiểu C
System.out.println("Ten: %s", ten);

// ✅
System.out.printf("Ten: %s%n", ten);
System.out.println("Ten: " + ten);
System.out.printf(String.format("Ten: %s", ten));
```

# `super`

`super` là từ khóa ám chỉ lớp cha, dùng trong lớp con kế thừa lớp cha.

`super()` là gọi constructor của lớp cha, và luôn phải đặt đầu tiên trong constructor của lớp con.
