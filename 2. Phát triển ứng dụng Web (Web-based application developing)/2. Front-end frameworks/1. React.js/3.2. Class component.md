
# Class component

Class component là render component bằng class thay vì function. Trong đó:
- Các state được thay bằng attribute.
- Các effect được thay bằng các method.

| Class Component        | Function Component        |
| ---------------------- | ------------------------- |
| `state`                | `useState`                |
| `componentDidMount`    | `useEffect`               |
| `componentDidUpdate`   | `useEffect`               |
| `componentWillUnmount` | cleanup trong `useEffect` |
| `Context`              | `useContext`              |
| `PureComponent`        | `React.memo`              |
| `Refs`                 | `useRef`                  |

**VD**:
```jsx
import React from "react";

class Counter extends React.Component {
  state = {
    count: 0,
  };

  increase = () => {
    this.setState({
      count: this.state.count + 1,
    });
  };

  render() {
    return (
      <div>
        <h1>{this.state.count}</h1>
        <button onClick={this.increase}>Increase</button>
      </div>
    );
  }
}
```

**Vì sao function component phổ biến hơn class component ?**
- Function component có cú pháp đơn giản dễ đọc hơn, không bị rơi vào các thủ tục rườm rà của OOP.
- Function component có hệ thống các hook có thể thay thế được class component methods, muốn dùng cái gì thì chỉ cần import vào và dùng.
