
# Giới thiệu & Cài đặt

**React Router DOM** là thư viện giúp xây dựng **Single Page Application (SPA)** bằng cách quản lý việc điều hướng (routing) giữa các trang ở phía client mà không cần reload toàn bộ trang web.

Cài đặt:

```bash
npm install react-router-dom
```

# Các thành phần cơ bản

## `BrowserRouter`

Bao bọc toàn bộ ứng dụng để kích hoạt cơ chế routing.

```jsx
import { BrowserRouter } from "react-router-dom";

function App() {
  return (
    <BrowserRouter>
      <Main />
    </BrowserRouter>
  );
}
```

## `Routes` và `Route`

`Routes` chứa danh sách các `Route`.

Mỗi `Route` ánh xạ một URL (`path`) với một component (`element`) sẽ được render khi URL khớp.

```jsx
<Routes>
  <Route path="/" element={<Home />} />
  <Route path="/about" element={<About />} />
</Routes>
```

## Nested Route và `Outlet`

Cho phép nhiều route dùng chung layout.

```jsx
// Route mặc định là <Home /> (index), tương đương với path "" hoặc "/"
<Routes>
  <Route path="/" element={<Layout />}>
    <Route index element={<Home />} />
    <Route path="about" element={<About />} />
  </Route>
</Routes>
```

Component cha:

```jsx
import { Outlet } from "react-router-dom";

function Layout() {
  return (
    <>
      <Header />
      <Outlet />
      <Footer />
    </>
  );
}
```

`<Outlet />` đại diện cho vị trí dùng để render route con tương ứng.

## Private Route

Là một pattern thường dùng để bảo vệ các trang yêu cầu đăng nhập.

**Cách 1**: Bọc route bằng `PrivateRoute`:

```jsx
<Route
  path="/dashboard"
  element={
    <PrivateRoute>
      <Dashboard />
    </PrivateRoute>
  }
/>
```
Handle `PrivateRoute`:
```jsx
function PrivateRoute({ children }) {
  const isAuthenticated = true;

  return isAuthenticated
    ? children
    : <Navigate to="/login" />;
}
```

**Cách 2**: Sử dụng nested routes:

```jsx
<Route element={<PrivateRoute />}>
  <Route path="/dashboard" element={<Dashboard />} />
</Route>
```
Handle `PrivateRoute`:
```jsx
import { Navigate, Outlet } from "react-router-dom";

function PrivateRoute() {
  const isAuthenticated = true; // ...

  return isAuthenticated
    ? <Outlet />
    : <Navigate to="/login" replace />;
  // Nếu chưa đăng nhập, người dùng sẽ được chuyển đến `/login`.
}
```

# Route Parameters

## `useParams()`

Dùng để lấy **path parameter** từ URL.

Khai báo route:

```jsx
<Route
  path="/users/:userId/posts/:postId"
  element={<PostDetail />}
/>
// Sample : /users/5/posts/20
```

Lấy tham số:
```jsx
import { useParams } from "react-router-dom";

function PostDetail() {
  const { userId, postId } = useParams();

  // userId = "5"
  // postId = "20"
}
```

**Lưu ý**:
- Giá trị trả về luôn là `string`.
- Nếu tham số không tồn tại thì giá trị là `undefined`.

## `useSearchParams()`

Dùng để lấy hoặc cập nhật **query parameter**. Query parameter không cần phải khai báo trong `Route`.

URL:
```text
/products?page=2&keyword=iphone
```

Lấy tham số:
```jsx
import { useSearchParams } from "react-router-dom";

function Products() {
  const [searchParams] = useSearchParams();

  const page = searchParams.get("page");
  const keyword = searchParams.get("keyword");
}
```

Cập nhật tham số:
```jsx
const [, setSearchParams] = useSearchParams();

setSearchParams({ page: 2, keyword: "iphone" });
```

**Lưu ý**
- `.get()` trả về `string | null`.
- Nếu parameter không tồn tại thì trả về `null`.

# Điều hướng (Navigation)

## `Link`

Điều hướng mà không reload trang. Tương tự thẻ `<a>` nhưng React Router sẽ xử lý việc chuyển trang phía client.

```jsx
import { Link } from "react-router-dom";

<Link to="/about">
  About
</Link>
```

## `NavLink`

Giống `Link` nhưng có khả năng nhận biết route hiện tại đang active thông qua thuộc tính `isActive` của nó.

**VD**:
```jsx
<NavLink
  to="/about"
  className={({ isActive }) => isActive ? "active" : "" }
>
  About
</NavLink>
```

## `Navigate`

Điều hướng bằng cách render component.

```jsx
import { Navigate } from "react-router-dom";

if (!user) {
  return <Navigate to="/login" replace />;
}
```

Thường dùng cho:
- `PrivateRoute`.
- Redirect sau khi kiểm tra quyền truy cập

## `useNavigate()`

Điều hướng bằng code.

**VD1**: Điều hướng thường:
```jsx
import { useNavigate } from "react-router-dom";

function Login() {
  const navigate = useNavigate();

  const handleLogin = () => {
    // ...
    navigate("/dashboard");
  };

  return (
    <button onClick={handleLogin}>
      Login
    </button>
  );
}
```

**VD2**: Điều hướng không thêm vào lịch sử duyệt web:
```jsx
navigate("/dashboard", { replace: true, });
```

**VD3**: Undo page.
```jsx
navigate(-1);
```

**VD4**: Redo page.
```jsx
navigate(1);
```
