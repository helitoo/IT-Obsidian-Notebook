
**GSAP (GreenSock Animation Platform)** là một thư viện mạnh mẽ cho phép thực hiện nhiều hiệu ứng phức tạp trên web. Bạn có thể tích hợp nó vào dự án React thông qua **GSAP React**.

```sh
npm install gsap @gsap/react
```

>[!important]
>Trường hợp các thuộc tính CSS liên quan đến transition, animation,... của 1 object cùng bị tác động bởi GSAP, TailwindCSS hoặc các library khác có thể dẫn đến việc xung đột và hiệu ứng không được triển khai. Tốt nhất chỉ nên dùng một cái. 

# Tween

**Tween** là các phương thức có chức năng *thay đổi trạng thái* của object. Tức là nó animate object.

Chỉ có 3 tween:
- **`.to()`**: Thay đổi trạng thái của object từ trạng thái gốc thành trạng thái được `.to()` chỉ định.
- **`.from()`**: Thay đổi trạng thái của object từ trạng thái được `.from()` chỉ định thành trạng thái gốc.
- **`.fromTo()`**: Thay đổi trạng thái của object từ trạng thái này sang trạng thái kia, đều định nghĩa trong phương thức này.

**Vị trí sử dụng tween**: Sử dụng bên trong `useGSAP` hoặc `useEffect`:

| Tiêu chí                                     | `useEffect`               | `useGSAP`    |
| -------------------------------------------- | ------------------------- | ------------ |
| Tạo animation GSAP                           | ✅                         | ✅            |
| Tự động cleanup animation                    | ❌ phải tự viết            | ✅ tự động    |
| React Strict Mode                            | ⚠️ dễ tạo animation 2 lần | ✅ xử lý sẵn  |
| Giới hạn phạm vi hoạt động của tween (scope) | ❌ không có                | ✅ có `scope` |
| Revert animation khi unmount                 | ❌ tự làm                  | ✅ tự động    |
| Viết code                                    | Dài hơn                   | Ngắn hơn     |

**VD**: Di chuyển `#box` sang vị trí `250px` (chiều dương).
```jsx
gsap.to('#box', { x: 250 })
```

**VD**: Di chuyển `#box` đến vị trí `250`, rồi quay về `0`.
```jsx
gsap.fromTo('#box', { x: 0 }, { x: 250 })
```

CSS selector này cũng có thể được thay bằng `ref` và `useRef()`.

| Thuộc tính               | Ý nghĩa                               | VD                                 |

| `rotationX`, `rotationY` | Xoay 3D                               | `rotationY: 180`                   |

| `scaleX`, `scaleY`       | Scale từng trục                       | `scaleX: 2`                        |
| `opacity`                | Độ trong suốt                         | `opacity: 0`                       |
| `width`, `height`        | Kích thước                            | `width: 300`                       |
| `backgroundColor`        | Màu nền                               | `backgroundColor: "#ff0000"`       |
| `color`                  | Màu chữ                               | `color: "white"`                   |
| `borderRadius`           | Bo góc                                | `borderRadius: "50%"`              |

### Các thuộc tính kiểm soát thời gian

| Thuộc tính        | Ý nghĩa                             | VD                       |
| ----------------- | ----------------------------------- | ------------------------ |
| `duration`        | Thời gian chạy (giây)               | `duration: 1.5`          |
| `delay`           | Trễ trước khi chạy                  | `delay: 0.5`             |
| `ease`            | Kiểu chuyển động                    | `ease: "power3.out"`     |
| `repeat`          | Số lần lặp, `-1` nếu là lặp vô hạn. | `repeat: 3`              |
| `repeatDelay`     | Trễ giữa các lần lặp                | `repeatDelay: 1`         |
| `yoyo`            | Chạy ngược khi lặp                  | `yoyo: true`             |
| `paused`          | Khởi tạo ở trạng thái pause         | `paused: true`           |
| `immediateRender` | Render ngay (hay dùng với from)     | `immediateRender: false` |

### Các thuộc tính kiểm soát sự kiện

|Thuộc tính|Khi nào chạy|
|---|---|
|`onStart`|Animation bắt đầu|
|`onUpdate`|Mỗi frame|
|`onComplete`|Hoàn thành|
|`onRepeat`|Mỗi lần repeat|
|`onReverseComplete`|Reverse xong|
|`onInterrupt`|Bị hủy giữa chừng|

### `ease`

| `ease`          | Cảm giác               |
| --------------- | ---------------------- |
| `"none"`        | Tốc độ đều             |
| `"power1.out"`  | Mượt nhẹ               |
| `"power2.out"`  | Mượt vừa               |
| `"power3.out"`  | Rất phổ biến           |
| `"power4.out"`  | Mạnh                   |
| `"back.out"`    | Vượt đích rồi quay lại |
| `"elastic.out"` | Đàn hồi                |
| `"bounce.out"`  | Nảy                    |
| `"expo.out"`    | Tăng/giảm tốc mạnh     |
| `"circ.out"`    | Chuyển động tròn       |


### `stagger`

Là thuộc tính kiểm soát animation của các object nếu selector của `.to()`,... là 1 tập hợp các object.

**VD**:
```jsx
stagger: {
  each: 0.1,
  from: "center",
  grid: [3, 3],
  ease: "power2.out"
}
```

Đối với object có nội dung và bạn muốn áp dụng stagger cho từng ký tự (`chars`) hoặc từng từ (`words`) của nội dung đó, thì phải dùng plugin `SplitText`.

```jsx
gsap.registerPlugin(SplitText);

// Begin the component

useGSAP(() => {
  const split = new SplitText("#title", { type: "chars" });
  gsap.fromTo(split.chars, {...}, {...});
});
```

| Thuộc tính | Ý nghĩa                      | VD                     |
| ---------- | ---------------------------- | ---------------------- |
| `each`     | Khoảng cách giữa các phần tử | `each: 0.1`            |
| `amount`   | Tổng thời gian stagger       | `amount: 1`            |
| `from`     | Điểm bắt đầu stagger         | `from: "center"`       |
| `grid`     | Bố cục lưới                  | `grid: [3,3]`          |
| `axis`     | Chỉ stagger theo trục        | `axis: "x"`            |
| `ease`     | Ease áp dụng cho stagger     | `ease: "power2.inOut"` |
| `repeat`   | Lặp riêng stagger            | `repeat: 2`            |
| `yoyo`     | Chạy ngược stagger           | `yoyo: true`           |

| `from`     | Ý nghĩa              |
| ---------- | -------------------- |
| `"start"`  | Từ phần tử đầu tiên  |
| `"end"`    | Từ phần tử cuối      |
| `"center"` | Từ giữa              |
| `"edges"`  | Từ các cạnh vào giữa |
| `"random"` | Ngẫu nhiên           |
| `0`        | Phần tử index 0      |
| `3`        | Phần tử index 3      |

### `scrollTrigger`

Animation của object đăng ký thuộc tính này chỉ được kích hoạt khi object được chỉ định bởi thuộc tính này xuất hiện trên màn hình. Đây là 1 plugin, nên cần đăng ký nó trước khi sử dụng.

**VD**:
```jsx
import { ScrollTrigger } from "gsap/all";
gsap.register(ScrollTriger);

// Begin the component
const scrollRef = useRef(null); // Attach on the trigger

useGSAP(() => {
  const boxes = gsap.utils.toArray(scrollRef.current.children);
  boxes.forEach((box) => {
    gsap.to(
      box,
      scrollTrigger: {
        trigger: box,
        // ...
      }
    );
  })
}, {scope: scrollRef});
```

Một số thuộc tính của `scrollTrigger`:

| Thuộc tính      | Ý nghĩa                                        | VD                                                                                        |
| --------------- | ---------------------------------------------- | ----------------------------------------------------------------------------------------- |
| `trigger`       | Element kích hoạt                              | `trigger: ".section"`                                                                     |
| `start`         | Điểm bắt đầu                                   | `start: "top 80%"`<br>(*mép `top` của trigger chạm vào vị trí `80%` chiều cao viewport.*) |
| `end`           | Điểm kết thúc                                  | `end: "bottom top"`<br>(*mép `bottom` của trigger chạm vào mép `top` của viewport.*)      |
| `scrub`         | Liên kết animation với scroll                  | `scrub: true`                                                                             |
| `markers`       | Hiện marker debug                              | `markers: true`                                                                           |
| `pin`           | Ghim phần tử                                   | `pin: true`                                                                               |
| `toggleActions` | Điều khiển play/reverse                        | `toggleActions: "play none none reverse"`                                                 |
| `onEnter`       | Callback thực hiện khi scroll vào trigger.     |                                                                                           |
| `onEnterBack`   | Callback thực hiện khi scroll ra khỏi trigger. |                                                                                           |

**VD**: `scrub`, `start` và `end`:
```jsx
gsap.to(".box", {
  x: 500,
  scrollTrigger: {
    trigger: ".box",
    start: "top center",
    end: "bottom top",
    scrub: true
  }
});
```
Lúc này:
- `start` = animation progress = 0%
- `end` = animation progress = 100%
GSAP lấy khoảng scroll giữa `start` và `end` để map thành tiến độ animation.


# `gsap.timeline()`

Là phương thức cho phép điều khiển trình tự của các tween.

**VD**:
```jsx
const timeline = gsap.timeline();

useGSAP(() => {
  // Di chuyển sang phải, đến x = 250
  timeline.to('#box', {x: 250});
  
  // Di chuyển xuống dưới, đến y = 250
  timeline.to('#box', {y: 250});
  
  // Di chuyển sang phải, đến x = 500
  timeline.to('#box', {x: 500})
})
```

**VD**: Kích hoạt hoặc dừng lại timeline:
```jsx
<button onClick={() => {
  if (timeline.paused()) timeline.play();
  else timeline.paused();
}}>
  Play/Paused
</button>
```

### Các thuộc tính của `.timeline()`

| Thuộc tính           | Ý nghĩa                                       | VD                                              |
| -------------------- | --------------------------------------------- | ----------------------------------------------- |
| `defaults`           | Giá trị mặc định cho mọi tween trong timeline | `defaults: { duration: 1, ease: "power2.out" }` |
| `delay`              | Trễ trước khi timeline bắt đầu                | `delay: 1`                                      |
| `paused`             | Khởi tạo ở trạng thái dừng                    | `paused: true`                                  |
| `repeat`             | Số lần lặp                                    | `repeat: 3`                                     |
| `repeatDelay`        | Thời gian nghỉ giữa các lần lặp               | `repeatDelay: 0.5`                              |
| `yoyo`               | Chạy ngược sau mỗi lần lặp                    | `yoyo: true`                                    |
| `smoothChildTiming`  | Đồng bộ timing khi thay đổi timeline          | `smoothChildTiming: true`                       |
| `autoRemoveChildren` | Tự xóa tween đã hoàn thành                    | `autoRemoveChildren: true`                      |
| `onStart`            | Callback khi timeline bắt đầu                 | `onStart: fn`                                   |
| `onUpdate`           | Callback mỗi frame                            | `onUpdate: fn`                                  |
| `onComplete`         | Callback khi hoàn thành                       | `onComplete: fn`                                |
| `onRepeat`           | Callback khi repeat                           | `onRepeat: fn`                                  |
| `onReverseComplete`  | Callback khi reverse xong                     | `onReverseComplete: fn`                         |

### Position parameter

**Position parameter** là đối số thứ 3 của các tween dùng để xác định *animation sẽ được chèn vào đâu trong timeline*.

| Position parameter | Ý nghĩa                                                 |
| ------------------ | ------------------------------------------------------- |
| `"+=1"`            | Bắt đầu **sau** vị trí hiện tại 1s                      |
| `"-=1"`            | Bắt đầu **sớm** hơn vị trí hiện tại 1s                  |
| `0`                | Bắt đầu **tại** giây thứ 0                              |
| `1.5`              | Bắt đầu tại giây thứ 1.5                                |
| `"<"`              | **Cùng thời điểm bắt đầu** animation trước              |
| `">"`              | **Cùng thời điểm kết thúc** animation trước \[Mặc định] |
| `"<+=1"`           | **Sau** khi animation trước **bắt đầu** 1s              |
| `"<-=1"`           | **Trước** khi animation trước **bắt đầu** 1s            |
| `">+=1"`           | **Sau** khi animation trước **kết thúc** 1s             |
| `">-=1"`           | **Trước** khi animation trước **kết thúc** 1s           |

**VD**:
```jsx
tl
.to("#box1", { duration: 1 })
.to("#box2", { duration: 1 }, "+=0.5");
```
- Ban đầu, `#box2` sẽ bắt đầu tại giây thứ 1.
- Nhờ có `"+=0.5"`, nó sẽ bắt đầu trễ hơn `0.5`s, cụ thể là tại giây 1.5.
